安装CUDA

约 296 字小于 1 分钟

电脑AI

2022-11-30

1.下载 CUDA 安装包

下载链接选择最新版

选择 exe 下载安装. 打开之后会先让你解压，再会开始安装.

一路确认即可.

等待安装完成.

2.确认是否安装完成

2.1 确认环境变量

打开系统变量 ( 设置->系统->关于->高级系统设置 )

查看是否存在以下变量.

2.2 确认是否可用

按下 WIN+R，打开 cmd，输入命令 ( 注意，必须是大写 V )

nvcc -V

2.3 如果无法输出版本号

打开 PATH 系统变量

如果没有请按照你的安装路径补全! 补全后再尝试执行命令!

3.安装 cuDNN

3.1 下载软件

下载链接

选择对应的版本

下载的话需要注册为英伟达会员资格，也不用钱，注册一下就 ok

3.2 下载完成

打开安装路径，解压压缩包内的文件.

如果提示覆盖就覆盖就可以了
